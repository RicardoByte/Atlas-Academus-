___

### 1. Gestão de Alunos

- **RF-ATA-001**: O sistema deve permitir o cadastro de alunos com dados pessoais (nome, CPF, RG, data de nascimento, endereço, telefone, e-mail)
- **RF-ATA-002**: O sistema deve registrar informações de saúde dos alunos (tipo sanguíneo, alergias, condições médicas, restrições físicas)
- **RF-ATA-003**: O sistema deve permitir a atualização e consulta de dados cadastrais dos alunos
- **RF-ATA-004**: O sistema deve registrar fotos e documentos dos alunos
- **RF-ATA-005**: O sistema deve manter histórico de frequência dos alunos
- **RF-ATA-006**: O sistema deve permitir a inativação/reativação de cadastros de alunos

### 2. Gestão de Planos e Mensalidades

- **RF-ATA-007**: O sistema deve cadastrar diferentes tipos de planos (mensal, trimestral, semestral, anual)
- **RF-ATA-008**: O sistema deve registrar as modalidades incluídas em cada plano (musculação, natação, crossfit, etc.)
- **RF-ATA-009**: O sistema deve controlar as matrículas dos alunos nos planos
- **RF-ATA-010**: O sistema deve gerar mensalidades automaticamente conforme o plano contratado
- **RF-ATA-011**: O sistema deve registrar pagamentos de mensalidades com data, valor e forma de pagamento
- **RF-ATA-012**: O sistema deve identificar mensalidades em atraso
- **RF-ATA-013**: O sistema deve permitir o registro de descontos e promoções

### 3. Gestão de Treinos

- **RF-ATA-014**: O sistema deve cadastrar fichas de treino para cada aluno
- **RF-ATA-015**: O sistema deve registrar exercícios com séries, repetições, carga e observações
- **RF-ATA-016**: O sistema deve associar treinos a personal trainers responsáveis
- **RF-ATA-017**: O sistema deve permitir a consulta do histórico de treinos do aluno
- **RF-ATA-018**: O sistema deve registrar avaliações físicas periódicas (peso, altura, IMC, percentual de gordura, medidas)

### 4. Gestão de Funcionários

- **RF-ATA-019**: O sistema deve cadastrar funcionários com dados pessoais e profissionais
- **RF-ATA-020**: O sistema deve registrar os cargos/funções (recepcionista, personal trainer, instrutor, gerente)
- **RF-ATA-021**: O sistema deve controlar os horários de trabalho dos funcionários
- **RF-ATA-022**: O sistema deve registrar as qualificações e certificações dos profissionais

### 5. Gestão de Aulas

- **RF-ATA-023**: O sistema deve cadastrar aulas em grupo (spinning, pilates, yoga, dança, etc.)
- **RF-ATA-024**: O sistema deve controlar a agenda de aulas com horários, instrutores e salas
- **RF-ATA-025**: O sistema deve permitir reserva de vagas em aulas
- **RF-ATA-026**: O sistema deve limitar o número de participantes por aula
- **RF-ATA-027**: O sistema deve registrar a presença dos alunos nas aulas

### 6. Controle de Acesso

- **RF-ATA-028**: O sistema deve registrar entradas e saídas dos alunos na academia
- **RF-ATA-029**: O sistema deve validar se o aluno está com mensalidade em dia no momento do acesso
- **RF-ATA-030**: O sistema deve gerar relatórios de frequência por período

### 7. Gestão de Equipamentos

- **RF-ATA-031**: O sistema deve cadastrar equipamentos da academia
- **RF-ATA-032**: O sistema deve registrar manutenções preventivas e corretivas
- **RF-ATA-033**: O sistema deve controlar o status dos equipamentos (disponível, em manutenção, fora de uso)

### 8. Relatórios e Consultas

- **RF-ATA-034**: O sistema deve gerar relatórios financeiros (receitas, inadimplência, previsões)
- **RF-ATA-035**: O sistema deve gerar relatórios de alunos ativos/inativos
- **RF-ATA-036**: O sistema deve gerar relatórios de ocupação de aulas
- **RF-ATA-037**: O sistema deve permitir consultas personalizadas ao banco de dados
